== Description ==
Virtual Pay Payment Gateway plugin help making payments on wordpress (woocommerce) easier, by using the Virtual Pay pament gateway API. Accept payments via card and mobile money payments

== Installation ==

= Automatic Installation =

1. In the dashboard navigate to `Plugins > Add New`
2. Search for `Virtual Pay Payment Gateway`
3. Locate the `Virtual Pay Payment Gateway` by `Virtual Pay` and click `Install Now`
4. After installation, click on `Activate Plugin` to activate

= Manual Installation =

1. Upload `Virtual Pay Payment Gateway` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin settings page and configure as required

== Frequently Asked Questions ==

= What does this plugin needs to work ? =

This is a plugin for woocommerce, so it works if woocommerce is installed and activated.

= Can I use different modes of payment using this plugin? =
Yes. You can use Amex, Mastercard, Visa, Union Pay, Mpesa, Tigo Pesa and MTN.
